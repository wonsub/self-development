1. 등호(=)는 지금 입력하는 것이 수식이라는 것을 알림
2. 함수 또한 수식의 일부분이므로 이 역시도 등호로 시작되어야 함.
3. 수식은 상수,연산자, 셀 주로, 함수 등의 조합으로 구성
4. 수식 입력 도중 마우스로 셀을 클릭하면 수식에 셀 주소가 입력됨.
5. 수식을 복사할려면 셀 모서리에 마우스 포인터를 올려놓으면 십자 모양의 포인터가 나오는데 이를 드래그하면 가능.
6. 상대주소 = 수식을 복사했을 때 자동으로 행 번호나 열 문자가 변경되는 셀 주소
7.서식을 바꾸지 않고 수식만 복사하기
 7-1. 채우기 핸들을 드래그해서 수식을 복사하면 (자동 채우기 옵션)버튼이 생성되는데 여기서 [서식 없이 채우기]를 클릭하면 됨
 7-2. 수식을 복사할 셀 범위를 블록으로 지정한 다음 수식을 입력하고 ctrl+enter를 누름
8.연산자
 8-1.산술 연산자: 기본적인 수학 연산을 실행
 	+더하기,-빼기,*곱하기,/나누기,^거듭제곱,%백분율
 8-2.비교 연산자: 두 값을 비교하여 참 또는 거짓으로 나타냄
	>크다,<작다,>=이상,<=이하,=같다,<>같지 않다.
 8-3.텍스트 연결자: 좌우의 값을 하나의 값으로 연결
 8-4.참조 연산자. 계산에 필요한 셀 범위를 결합함
 	콜론(:) 두 참조 사이의 모든 셀에 대한 참조	k5:m10 ->[k5]셀부터 [m10]셀까지의 모든 셀 참조
	쉼표(,) 여러 참조를 하나의 참조로 결합		a3,b2,k7 ->[a3]셀 [b2]셀,[k7]셀 참조
	공백() 두 개의 참조에서 공통되는 셀에 대한 참조	a1:c10 c3:f7 -> [a1:c10]과 [c3:f7]의 공통 범위 [c3:c7]참조
 8-5.연산자 우선 순위 (왼쪽에서 오른쪽이며, 순서를 변경하고자 할때는 괄호 사용)
	참조 연산자| 콜론(:)->공백(),->쉼표()
	산술 연산자| 음수(-)->백분율(%),거듭제곱(^)->곱하기와 나누기(*,/)->더하기와 빼기(+,-)
	텍스트 연결 연산자| *
	비교 연산자	모든 비교 연산자의 우선순위는 같다.
9.수식 자동 완성을 사용을 활성화 할 시.  파일-옵션-excel옵션-수식-수식 작업-수식 자동 완성 사용을 선택
10. 어떤 함수를 사용해야 할 지 알 수 없을 때 함수 마법사를 통해 입력가능(수식탭->함수삽입->범주 선택)
11. 범주를 안다면 수식 탭에서 함수 라이브러리 그룹에서 골라 선택

12. 상대 참조는  수식을 다른 곳으로 복사하거나 이동했을 때 자동으로 셀 주소가 변경된다.
13. 워크시트의 다른 셀에 있는 값을 이용하여 계산할 때 수식에 셀 주소를 포함시키는 것을 '셀 참조'라 한다.(일반적으로 'a1'스타일로 상대참조라 부름)
14. 셀에서 수식 보기(ctrl + ' 또는 [수식]-[수식분석]그룹에서 [수식표시]) 또  누를시 원래 결과값 출력
15. 절대 참조($가 앞에 붙음)는 수식을 다른 곳으로 이동하거나 복사하더라도 참조 범위가 변하지 않음. 항상 고정된 셀을 참조할 때 사용.
16. 해당 셀을 입력 후 F4를 누르면 셀 주소가 절대 참조로 변경됨.
17. F4로 셀 참조 형태 바꾸기(순서대로 바뀜)
    A1 -> $A$1 -> A$1 -> $A1
   상대	  절대   혼합   혼합
18. 혼합참조 열 문자 앞에만 또는 행 번호 앞에만 사용하는 형태로 구분됨. $가 사용되지 않은 부분만 복사 위치에 따로 자동 조정.
19. 다른 시트,다른 통합 문서의 셀 참조
  19-1.다른 워크시트의 셀 참조: 셀 참조 앞에 워크시트 이름과 느낌표를 입력한다. ex) =매출!K5*80%
  19-2.다른 통합 문서의 셀 참조하기: 대괄호 안에 통합 문서 이름, 워크시트 이름을 입력하고 느낌표를 셀 참조 앞에 입력한다.
       ex)=[실적.xlsx]매출!K5*80%
  19-3.통합 문서 이름이나 워크시트 이름에 숫자나 기호 문자, 공백 등이 포함되어 있으면 통합 문서 이름과 워크시트 이름을 작은 따옴표로 묶어야 한다.
       es)='1사분기'!K5 또는 ='[1월매출.xlsx]실적'!K5 형태

20. 수식이나 상수등에 이름을 정의할 수 있다.
21. [이름 상자]는 현재 셀 포인터가 있는 셀의 셀 주소를 표시한다. 또 한 이를 이용해서 이름을 쉽게 정의할 수 있다.
22. [이름 상자]의 드롭다운 버튼을 통해 현재 정의 되어 이쓴 이름 목록을 확인하고, 목록에서 참조하는 셀이나 셀 범위를 선택할 수 있다.
23. 이름 만들기를 통해 블록으로 지정한 셀 범위의 특정 부분에 있는 텍스를 이름으로 지정할 수 있다. 
24. 이름 영역에 숫자가 입력되어 있으면 숫자 앞에 자동으로 언더 바가 추가된다.\
25. 이름 작성 규칙
 25-1.사용할 수 있는 문자: 문자,숫자,마침표,언더 바 등으로 구성한다. 첫 글자는 반드시 문자나 언더 바로 시작해야 한다.
 25-2.참조 형식: 기본적으로 절대 참조가 사용된다. 상대 참조로 이름을 정의할 수도 있다.
 25-3.영어 대소문자를 모두 사용할 수 있지만 대소문자를 구분하지는 않는다.
 25-4.공백을 포함시킬 수 없다. 'K77' 또는 'AB$3'등 셀 참조와 같은 형식은 사용할 수 없다.
 25-5.글자 수 제한: 최대 255 글자까지 입력할 수 있다.

26. 수식이 반환할 수 이쓴ㄴ 오류 값은 '####' 오류를 포함해서 모두 여덟 가지이다.
27. 수식의 오류 
 27-1. 셀의 너비가 좁은 경우(#### 오류)
 27-2. 날짜 또는 시간이 음수인 경우
 27-3. 수식을 0으로 나눈 경우
 27-4. 지정한 값을 찾지 못할 때
 27-5. 이름이 정의되지 않은 경우
 27-6. 함수의 이름을 제대로 안쓴 경우
 27-7. 교차하지 않는 영역의 논리곱을 수행한 경우
 27-8. 잘못된 숫자 값을 사용한 경우
 27-9. 인수,피연산자의 형식이 잘못된 경우
 27-10.셀 참조가 유효하지 않는 경우

28. 중첩 수식 단계별로 계산하기: [수식]-[수식 분석] 그룹에서 [수식 계산]에서 계산으로 단계별로 진행 가능
29. 참조되는 셀, 참조하는 셀로 추적 가능(연결선 제거로 선제거 가능)
  29-1.참조할 때 오류 발생시 파란색 화살표 선 대신 빨간색이, 다른 워크시트나 다른 통합 문서의 셀과 연결 될 시 아이콘까지 이어지는 검은색 화살표가 대신한다.
